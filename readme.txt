Create forms with multiple input fields and have your users pay before submission.

== Description ==

With this plugin you can create forms with multiple input fields that allows customers to input necessary information and have users click on pay beore submission.

FOR CHURCHES AND NGO’S
Setting the form payment amount to ‘0’ allows the user to input any custom payment.
The user will enter all specificied information and enter the amount to be paid and proceed to pay.

FOR SELLING ITEMS
Merchants can create a form that allows the user to specify the quantity of goods to be purchased.
on the form,set quantified to be ‘yes’.
Set the max quantity a user can buy.

With this PayWithBank3D  plugin, you will be able to accept the following payment methods in your wordpress website:

* __Mastercard__
* __Visa__
* __Verve__
* __Bank Account__


Features:

*   Accept payment via MasterCard,Visa and Verve Cards.
*   Seamless integration into any WordPress page or post

== Installation ==

= Minimum Requirements =

* A Bank3D account


= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Payment Forms of PayWithBank3D, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “PayWithBank3D” and click Search Plugins. Once you’ve found our payment plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our payment plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
